Level

Original piece names Rules use Verdant, Ember, Abyss (classic: Yellow, Red, Brown). Same counts: 12 each, 36 total.

HueMind

Fill the board — mind the forbidden patterns.

How to play

Goal: Place all 36 pieces on the 6×6 board without creating any forbidden pattern for this level.

  • First move: place any color on any empty cell.
  • After that: each new piece must touch at least one piece already on the board (side or corner).
  • You have 12 of each color. Forbidden patterns count only when all cells in the pattern are the same color.
  • If a forbidden pattern appears at any time, you lose.

Levels 1–26 use a built-in progression of pattern rules (official cards show the exact shapes). Use the level description for this build.
